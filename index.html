1. Tujuan

  1. Peserta didik dapat memahami dasar pemrograman server-side menggunakan HTML.
  2. Peserta didik mengetahui sintaks-sintaks dasar HTML.
  3. Peserta didik mampu menampilkan dokumen sederhana menggunakan HTML.

2. Dasar Teori

HTML (HyperText Markup Language) merupakan bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser. Browser seperti Google Chrome atau Firefox akan menampilkan kode HTML menjadi tampilan visual halaman web.

Beberapa Elemen HTML yang Sering Digunakan:

Tag Fungsi
<h1>–<h6>Judul atau heading
<p>Paragraf teks
<a>Hyperlink
<img>Menampilkan gambar
<hr>Garis pemisah horizontal
<br>Ganti baris
<b>, <i>, <u>Format teks miring, tebal, garis bawah
